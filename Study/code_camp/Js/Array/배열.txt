

순서가 있는 데이터 컬렉션을 저장할떄 사용
-
데이터 컬렉션이 논리적으로 정의된 규칙에 의해 나열된것

** 여러개의 데이터를 순서와 함꼐 저장하는 창고

[] 배열 -> index

index = 0 1 2 3 4 5
length = 1 2 3 4 5 

length propperty(속성) 을 가지게 된다.

---

Method 다양한 기능을 담은 명령어

배열의 메서드

 push()
 배열의 가장 뒤에 데이터를 추가해주는 Method
 ranking.push('push하고싶은 데이터')
 
 pop()
 배열 가장 끝에 있는 데이터를 제거하는 Method

 ranking.pop()

 indexOf()
 

 includes()
 배열이 포함되어있는지!!
 어떠한 데이터가 특정 배열에 존재하는지 여부에 따라 true,false
 true,false로 반환된다;;
 ranking.includes('있는지 확인하고 싶은 데이터')
